# 🤖 Telegram Бот для МЭШ/ЦДЗ (aiogram version)

Этот бот построен на современной архитектуре и готов к работе «под ключ».

## 🛠 Инструкция по запуску

### 1. Настройка токена бота
Убедитесь, что в файле `.env` указан ваш токен от BotFather:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

### 2. Запуск
**Вариант А: Обычный запуск**
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите бота: `python main.py`

**Вариант Б: Через Docker**
1. Соберите образ: `docker build -t dz-bot .`
2. Запустите контейнер: `docker run -d --name dz-bot-instance dz-bot`

### 3. Как получить данные
1. В боте нажмите кнопку **⚙️ Настройки**.
2. Перейдите по ссылке для вашего региона (Москва или МО).
3. После входа скопируйте длинный токен (`eyJ...`) и отправьте его боту.
4. Используйте кнопки **👤 Профиль** и **💻 Мои тесты** для просмотра данных.

## 📂 Структура проекта
- `main.py` — Точка входа.
- `handlers/` — Логика команд и сообщений.
- `services/` — Взаимодействие с API школы.
- `database/` — Хранение данных (SQLite).
